# Vakuumier-Werkstatt

Statischer Ratgeber-Blog zum Thema Vakuumieren von Lebensmitteln (Folien-/Balkenvakuumierer) — drei SEO-optimierte Artikel, gehostet via GitHub Pages.

**Live:** https://nichtagentur.github.io/vakuumier-werkstatt/

## Stack
- Pure HTML/CSS, keine Build-Tools
- Schema.org Article + FAQPage + WebSite
- Open Graph + Twitter Cards
- robots.txt + sitemap.xml
- Bilder: Gemini 2.5 Flash Image via OpenRouter

## Struktur
```
index.html               # Hub mit 3 Artikel-Teasern + FAQ
artikel/
  frische-verlaengern.html
  sous-vide-marinieren.html
  sparen-meal-prep.html
ueber-uns.html           # E-E-A-T Autoren-Page
impressum.html           # § 5 TMG / § 24 MedienG
datenschutz.html         # GitHub Pages, kein Tracking
assets/style.css
img/                     # KI-generierte Editorial-Fotos
```

## Quality-Rater-Guidelines (Google QRG / E-E-A-T)
- Klare Hauptintention pro Artikel, kein Clickbait
- Autoren-Bio + Pseudonym-Offenlegung
- Quellen für Sicherheits-Aussagen (BZfE, Statistik Austria)
- Impressum + Datenschutz erreichbar von jeder Seite
- Keine Affiliate-Links, keine Werbe-Kooperationen, keine Tracker

